# D2 Audit Checklist

## Status
Informative checklist aligned with `OHAS_STANDARD.md` v0.1.

## Baseline
- [ ] All relevant D1 checks have been completed

## Controlled Workflow
- [ ] Managed editor or controlled workstation evidence is present
- [ ] Session records are present
- [ ] Policy state is recorded
- [ ] Device or environment identifier is present
- [ ] Workflow restrictions are documented

## Import and Clipboard Controls
- [ ] Import behavior is controlled, logged, or prohibited according to policy
- [ ] Any external insertion events are explained
- [ ] Clipboard policy is documented where applicable

## Toolchain and Restrictions
- [ ] Declared tools match observed environment
- [ ] Prohibited generative tools are absent from the certified workflow evidence
- [ ] Network mode is documented where relevant

## Scope and Profile
- [ ] Selected profile is compatible with the observed workflow
- [ ] Scope boundaries are clear and testable

## Conclusion
- [ ] Evidence supports D2
- [ ] Evidence supports only D1
- [ ] Corrective action required
- [ ] Insufficient evidence
