# D1 Audit Checklist

## Status
Informative checklist aligned with `OHAS_STANDARD.md` v0.1.

## Work Identification
- [ ] Work identifier is present
- [ ] Title is present
- [ ] Scope of certification is defined
- [ ] Claimed profile is stated
- [ ] Claimed assurance level is D1

## Declaration
- [ ] Author declaration is present
- [ ] Declaration is attributable to the named author
- [ ] Final content hash is present

## Version History
- [ ] A version history or dated draft sequence is present
- [ ] Version sequence is chronologically coherent
- [ ] Draft evolution appears plausible
- [ ] No unexplained large discontinuities are observed, or they are explained

## Toolchain
- [ ] Basic toolchain declaration is present
- [ ] Declared tools are compatible with the selected profile

## Exceptions
- [ ] Exceptions are either absent or declared
- [ ] Declared exceptions are compatible with scope and profile

## Conclusion
- [ ] Evidence supports D1
- [ ] Evidence supports only D0
- [ ] Corrective action required
- [ ] Insufficient evidence
