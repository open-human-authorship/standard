# P2 — No AI in Text Creation, AI Allowed Outside Core Text

## Status
Draft profile for OHAS v0.1.

## Purpose
This profile is intended for works whose certified text is human-authored, while AI may be used in adjacent activities outside the certified text body.

## Scope Boundary
The certified scope must explicitly identify the protected text area, for example:
- full manuscript text excluding cover materials
- body text excluding marketing copy
- screenplay dialogue and directions excluding pitch materials

## Allowed
- All allowances in P1
- AI-assisted cover ideation
- AI-assisted marketing copy outside certified scope
- AI-assisted metadata generation outside certified scope
- AI-assisted layout experimentation outside certified scope

## Prohibited Within Certified Scope
- Any generative AI contribution to the certified text
- AI rewriting, expansion, summarization, or paraphrasing of the certified text
- AI-assisted translation used as certified text unless separately declared and excluded from scope

## Required Disclosures
- Clear statement of what is inside and outside scope
- Disclosure of any AI used outside scope
- Final manifest scope declaration

## Evidence Expectations
- Same as P1 for the certified text
- Scope documentation
- External-use disclosure log where applicable

## Typical Use Cases
- Traditional publishing workflows
- Books with AI-assisted promotional assets
- Hybrid publishing environments that keep the text itself human-authored
