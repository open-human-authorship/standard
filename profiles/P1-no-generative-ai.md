# P1 — No Generative AI

## Status
Draft profile for OHAS v0.1.

## Purpose
This profile is intended for works that claim no use of generative AI in the creation of the certified text.

## Allowed
- Basic file saving and backup
- Non-generative spell checking
- Formatting and layout tools
- Accessibility tools that do not generate or rewrite text
- Bibliographic lookup and non-generative search tools

## Restricted / Requires Explicit Policy Decision
- Grammar tools with rewrite capability
- Speech-to-text systems with semantic correction
- OCR of handwritten author notes
- Controlled import of quotations and references

## Prohibited
- Prompt-based text generation
- Paraphrasing systems
- Autocomplete beyond minimal mechanical completion
- AI summarization used as source text for the certified work
- AI translation used directly as certified text
- AI-assisted plot, scene, paragraph, or sentence generation

## Evidence Expectations
- Author declaration
- Toolchain declaration
- Version history and session records appropriate to the assurance level
- Logged exceptions, if any

## Typical Use Cases
- Human-only literary works
- Competition submissions
- Archival or prestige publishing workflows
