# Roadmap

## Status
Illustrative planning document.

## Near-Term Goals
- stabilize `OHAS_STANDARD.md`
- refine profiles `P1` through `P5`
- review schema design
- improve examples
- collect public feedback from authors, publishers, educators, and auditors

## Medium-Term Goals
- define stronger manifest and record schemas
- add conformance language for issuers
- document accessibility accommodations
- define public status and revocation vocabulary
- publish implementation guidance for managed editors or secure writing environments

## Longer-Term Goals
- publish a versioned governance model
- explore interoperable credential formats
- pilot institutional and publishing workflows
- develop open-source tooling aligned with OHAS
- evaluate whether a future certification program is desirable

## Release Philosophy
Changes should aim for clarity, interoperability, and honest assurance rather than overstated certainty.
